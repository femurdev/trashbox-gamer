# Trashbox-Gamer

Have some downtime in your classes after AP exams?
Grab a Raspberry pi, an SD card, and a few friends and come game.
Supports up to 4 players, one of which can play on a keyboard. (hopefuly)
Native built to to use PS4 DualShock 4 Controllers.
